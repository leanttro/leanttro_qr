Como funciona o painel

  • + Criar novo QR code — gera um cartão fidelidade novo, com link e QR code próprios pra um cliente/unidade. Limite de {{ limite_qr_codes }} QR codes por conta.
  • Editar (em cada linha) — muda título, mensagem, meta de carimbos e prêmio daquele cartão específico{% if templates_gamificacao|length > 1 %} (e o modelo visual, se você tiver mais de um){% endif %}.
  • +1 / Zerar — carimba (ou zera) o cartão na mão, direto do painel, sem precisar abrir o link do cliente.
  • Baixar PNG — baixa a imagem do QR code pra imprimir e colar no balcão.
  • Abrir / Compartilhar — abre o link do cartão ou manda pro cliente (WhatsApp, etc.).
  • Salvar link — troca só a parte final do endereço; o começo continua fixo com o nome do seu negócio.
  • Sortear um QR code — sorteia aleatoriamente um dos seus cartões, útil pra escolher uma unidade ou destaque do mês.
